# 星座运势手机应用

一个功能完整的星座运势查询应用，支持手机和电脑端访问，可以分享给其他人使用。

## 🌟 功能特性

- **12星座运势查询**：支持所有12个星座的运势分析
- **全方位运势分析**：包含整体、爱情、事业、财富、健康运势
- **星级评分系统**：5星制直观显示运势好坏
- **今日建议**：提供实用的生活建议
- **本地存储**：记住用户选择的星座
- **响应式设计**：完美适配手机和电脑端
- **离线可用**：纯前端应用，无需后端支持

## 🚀 快速开始

### 本地运行
```bash
# 方法1：使用Python（推荐）
python -m http.server 8080

# 方法2：使用Node.js
npm install -g http-server
http-server -p 8080

# 方法3：使用PHP
php -S localhost:8080
```

### 访问应用
打开浏览器访问：`http://localhost:8080`

## 🌐 网络部署

### 方法一：GitHub Pages（免费）
1. 在GitHub创建新仓库
2. 上传所有文件到仓库
3. 在仓库设置中启用GitHub Pages
4. 访问 `https://你的用户名.github.io/仓库名`

### 方法二：Vercel（免费）
1. 注册Vercel账号
2. 连接GitHub仓库
3. 一键部署
4. 获得专属域名

### 方法三：Netlify（免费）
1. 注册Netlify账号
2. 拖拽文件夹上传
3. 自动部署
4. 自定义域名支持

### 方法四：腾讯云/阿里云静态网站托管
1. 购买云服务
2. 上传文件到对象存储
3. 配置CDN加速
4. 绑定自定义域名
